# Ritvik Jaiswal — Portfolio

Personal portfolio website built with **Next.js 14 (App Router)**, focused on typography, motion, and performance.

---

## Tech

* Next.js 14
* TypeScript
* CSS Modules + CSS Variables
* Framer Motion, GSAP
* @fontsource (Instrument Serif, DM Sans, Inter)

---

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

---

## Build

```bash
npm run build
npm start
```

---

## Customization

* Content and layout: `app/components`
* Global styles and theme: `app/globals.css`
* Fonts: `app/layout.tsx`
* Assets: `public/`

---

## Deployment

Optimized for deployment on Vercel.
